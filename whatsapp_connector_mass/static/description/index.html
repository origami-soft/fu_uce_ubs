Batch/Group messaging

Filter > Group > Select records in list views

Filter > Group > Select:

1) Open list views.

2) Apply your filters.

3) Select all or some records.


Action Menu

4) Click on Action menu.

5) Select Send Multi Whatsapp.


Message options

Reference:
Short description.

Send by:
Select from which Whatsapp number the messages will be sent (Module supports multi Whatsapp).

Put in queue:
Queue sending immediately or create in draft for advanced editing.

Unique contact:
Verify to send only one message per number (Do not repeat).

Template:
Select template for dynamic messages (Text and attachments).


View Batch/Group status

Go to Whatsapp Batch main menu.

Click on each Batch for statistics and details of each message.


Create groups of contacts to reuse

Go to Whatsapp Batch main menu.
